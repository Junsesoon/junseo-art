# 🗽프로젝트 개요
* initial date: 2026-06-13
* summary: 이 프로젝트의 목적은 개발자 오준서 라는 사람의 실력을 투명하게 보여드리고 다양한 영감을 얻어가실 수 있도록 3차원 인터랙션이 넘쳐나는 동적 웹을 구현하는 것 입니다
* concept: 명예의 전당
* content: 캐릭터가 이곳 저곳을 탐방하며 각종 프로젝트 및 다양한 인터랙션을 경험할 수 있게 만들 예정입니다

# 📂 폴더 구조 및 역할 (Project Structure)
이 프로젝트는 개발과 디자인 과정에서의 효율성을 높이기 위해 소스 코드와 에셋의 역할을 명확히 분리하여 관리합니다

* **`src/`**
  * **역할**: 프론트엔드 애플리케이션의 핵심 소스 코드입니다
  * **내용**: 3D 렌더링 로직(Three.js), 물리 엔진 연산, 게임 루프, 카메라 컨트롤, UI 로직 및 메인 진입점(`index.html`) 등의 JavaScript/HTML 코드가 위치합니다

* **`scripts/`**
  * **역할**: 프로젝트의 빌드, 배포, 에셋 최적화 등을 돕는 유틸리티 스크립트 모음입니다
  * **내용**: `asset-raw`의 무거운 원본 파일들을 웹용으로 가볍게 압축하고 변환하는 스크립트(`compress.js` 등)와 같은 Node.js 자동화 툴들이 포함됩니다

* **`asset-raw/`**
  * **역할**: 가공되기 전의 디자인/3D 아트 원본(Raw) 파일 보관소입니다. 웹 브라우저에 직접 제공(Serving)되지 않습니다
  * **내용**: 3D 모델링 원본 파일(`Blender / .blend`), 텍스처 작업 파일(`Photoshop / .psd`, `Substance / .sbs`), 원본 사운드 등 프로덕션 작업용 파일들이 포함됩니다

* **`asset-static/`**
  * **역할**: 웹 브라우저가 직접 읽어 들이는 최적화된 정적(Static) 에셋 폴더입니다
  * **내용**: `asset-raw/`에서 작업 후 내보내기(Export) 및 압축된 웹 최적화 3D 모델(`.glb`), 텍스처 이미지(`.png`, `.jpg`, `.webp`), 사운드 파일 등으로 구성됩니다
