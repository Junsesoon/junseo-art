# 🗽프로젝트 개요
* initial date: 2026-06-13
* summary: 이 프로젝트의 목적은 개발자 오준서 라는 사람의 실력을 투명하게 보여드리고 다양한 영감을 얻어가실 수 있도록 3차원 인터랙션이 넘쳐나는 동적 웹을 구현하는 것 입니다
* concept: 명예의 전당
* content: 캐릭터가 이곳 저곳을 탐방하며 각종 프로젝트 및 다양한 인터랙션을 경험할 수 있게 만들 예정입니다

# 🚀실행 방법
## 1. 패키지 설치
프로젝트를 처음 클론(또는 다운로드)한 후, 개발 환경에 필요한 패키지들을 설치합니다.
* `npm install`

## 2. 에셋 최적화 (필수)
무거운 원본 3D 모델(`asset-raw`)을 웹용으로 자동 압축하여 `public/asset-static`에 저장합니다. **(최초 1회 또는 원본 모델 변경 시 반드시 실행해야 합니다)**
* `npm run compress`

## 3. 로컬 개발 서버 실행
실시간으로 변경사항을 확인할 수 있는 로컬 개발 서버를 엽니다.
* `npm run dev`

## 4. 빌드 및 배포 미리보기 (선택)
실제 배포 시 사용할 최적화된 결과물을 생성하고, 서비스 환경과 동일하게 로컬에서 테스트합니다.
* `npm run build`: 배포용 정적 파일 생성
* `npm run preview`: 생성된 빌드 결과물로 로컬 미리보기 서버 실행

# 📂 폴더 구조 및 역할 (Project Structure)
이 프로젝트는 개발과 디자인 과정에서의 효율성을 높이기 위해 소스 코드와 에셋의 역할을 명확히 분리하여 관리합니다

* **`src/`**
  * **역할**: 프론트엔드 애플리케이션의 핵심 소스 코드 보관소입니다
  * **내용**: 3D 컴포넌트(`components/`), 물리 연산(`physics/`), 압축 스크립트(`scripts/`) 및 메인 로직(`App.jsx`) 등이 위치합니다

* **`public/` (내부 `asset-static/`)**
  * **역할**: 웹 브라우저가 직접 읽어 들이는 최적화된 정적(Static) 파일 보관소입니다
  * **내용**: 스크립트로 압축된 최적화 3D 모델(`-opt.glb`) 등이 저장되어 서빙됩니다

* **주요 파일**
  * `index.html`: 프로젝트의 최초 진입점이자 빈 도화지 역할을 하는 최상위 마크업 파일입니다
  * `vite.config.js` / `package.json`: 빌드, 패키지, 스크립트 등 프로젝트 환경을 설정합니다
