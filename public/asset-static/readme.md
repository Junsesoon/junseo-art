# 📂 asset-static
* modified at: 2026-06-15

* 이 폴더는 웹 브라우저가 직접 읽어 들이는 **최적화된 정적(Static) 에셋**을 보관하는 용도로 사용됩니다
* `asset-raw/` 폴더에 있는 원본 3D 모델을 `npm run compress` 스크립트로 변환(압축)하면 이곳에 저장됩니다
* ⚠️스크립트로 자동 생성된 파일(`-opt.glb` 등)을 이곳에서 직접 수정하지 마세요. 수정이 필요하다면 `asset-raw/`의 원본을 수정한 후 다시 압축 스크립트를 실행해야 합니다

# files
* (여기에 스크립트로 자동 생성된 최적화 모델 외에, 파비콘이나 일반 이미지 등 직접 추가한 정적 파일이 있다면 기록합니다)
